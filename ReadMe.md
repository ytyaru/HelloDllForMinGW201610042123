# このソフトウェアについて #

HelloDllForMinGW201610042123は私個人が学習目的で作成したソフトウェアである。

MinGWのg++でライブラリを作成し呼び出した。

# 開発環境 #

* Windows XP Pro SP3 32bit
* [MinGW 5.3.0](http://ytyaru.hatenablog.com/entry/2016/10/07/100000)
    * g++ 5.3.0
